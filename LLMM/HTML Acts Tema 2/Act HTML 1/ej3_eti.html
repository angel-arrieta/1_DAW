<xmp>: dibuja texto entre las etiquetas de inicio y fin sin interpretar <br> el HTML que se encuentra en medio y lo muestra usando un tipo de letra monoespaciada < mark >: resalta el texto que tiene anidado
< addres >: el texto anidado se muestra como una dirección. Ejemplo:
xblanco22x@gmail.com 
            < pre >: El texto en este elemento típicamente se muestra en una fuente fija, no 

            proporcional, exactamente como es mostrado en el archivo. Ejemplo: 

            body{
                color:  red;
              }
              a   {
                color:green;
              }
< abbr >: Cuando incluya una abreviatura o un acrónimo, proporcione una expansión
completa del término en texto sin formato en el primer uso, junto con el < abbr >
para marcar la abreviatura. Esto informa al usuario qué significa la abreviatura
o sigla. Ejemplo:

Puedes usar CSS (Cascading Style Sheets) para estilizar tu HTML
(Lenguaje de Marca HyperTexto). Usando hojas de estilo, puedes mantener tu capa de
presentación CSS y tu capa de contenidos HTML separada.
Esto se llama "separacion de referencia."

< bdo >: Anula la direccionalidad actual del texto, de modo que el texto que
contiene se representa en una dirección diferente. Ejemplo: HOLA
< p >: Indica cuando un texto debe mostrarse como parrafo
< br >: Indica cuando debe haber un salto de línea (no necesita etiqueta de final)
< q >: Indica una cita corta. Ejemplo: ayer me caí
< meta>: Sirve para aportar información sobre el documento (no necesita etiqueta de final)
