Posicionamiento de objetos

Muy probablemente debido a la naturaleza de z-index,
este texto no se vera, justamente enfrente, habra otra imagen
apuntando hacia aquí, ya que este texto sera bloqueado
por la imagen de un flotador, si es que ha salido bien.
Mediante el uso de propiedas como display, align-items,
y position se pueden colocar los objetos de diversas
maneras diferentes. También con el uso de z-index
y position: absolute, se pueden apilar
elementos. De estas maneras puedes colocar
objetos en un lado u el otro
inclusive en el centro
